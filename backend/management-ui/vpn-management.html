üîê SAML VPN Authentication Guide

üìã What is SAML Authentication?

SAML (Security Assertion Markup Language) authentication requires you to authenticate through your organization's web portal before connecting to the VPN.

üöÄ Connection Process

  1. Get SAML URL: Click "üîó Get SAML Login URL" to retrieve the authentication URL
  2. Open Browser: Navigate to the SAML login URL in your browser
  3. Authenticate: Log in with your work credentials (may include 2FA)
  4. Connect: Once authenticated, attempt VPN connection
  5. Verify: Check connection status and test access to work resources

üîß Alternative Methods

OpenVPN Connect App

For the most reliable SAML authentication, use the official OpenVPN Connect app:

  • Download from: openvpn.net/downloads/
  • Import your .ovpn configuration file
  • The app will handle SAML authentication automatically

OpenVPN3 Command Line

Import config:
openvpn3 config-import --config /path/to/work.ovpn

Start session:
openvpn3 session-start --config work

Check status:
openvpn3 sessions-list

  • Browser doesn't open: Manually navigate to the SAML URL
  • Authentication fails: Check credentials and 2FA
  • Connection times out: Complete authentication within the time limit
  • Corporate firewall: VPN traffic may be blocked
